Decimals Questions

1. Define what a decimal is and explain how it relates to fractions.
Answer: A decimal is a way to represent fractions where the denominator is 10, 100, 1000, etc. It uses a point to separate whole and fractional parts.
2. Convert the following fractions to decimals: \( \frac{3}{10}, \frac{7}{100}, \frac{21}{1000} \).
Answer: 0.3, 0.07, 0.021
3. Write 5.4, 0.25, and 13.07 in expanded form.
Answer: 5.4 = 5 + 0.4; 0.25 = 0.2 + 0.05; 13.07 = 10 + 3 + 0.07
4. Place these decimals in ascending order: 3.45, 3.405, 3.5, 3.04.
Answer: 3.04, 3.405, 3.45, 3.5
5. What is the place value of the digit 7 in 45.726?
Answer: The place value of 7 is 0.7 (tenths place).
